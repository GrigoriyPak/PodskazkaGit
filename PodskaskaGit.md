# Подстказка по GIT

Создание репозитория:
```sh
git init
```

Добавление в репозиторий:
```sh
git add
```

Запись в репозиторий с комментариями:
```sh
git commit -m "Message"
```

```sh
git log
```

```sh
git log --oneline
```

```sh
git checkout
```

```sh
git checkout master
```

```sh
git diff
```

```sh
git status
```
